# Quick Start: Getting Blog Images

You have 3 options to get the 8 required blog images quickly:

---

## Option 1: AI Generation (Fastest - 30 minutes)

### Using Midjourney (Recommended)
1. Go to [midjourney.com](https://midjourney.com)
2. Join their Discord server
3. Use the prompts from `PROMPTS.md`
4. Generate each image with `/imagine [prompt]`
5. Download at 1200x630px
6. Optimize with TinyPNG.com

**Cost:** $10/month subscription  
**Time:** 30 minutes for all 8 images  
**Quality:** Excellent, unique, on-brand

### Using DALL-E 3 (via ChatGPT Plus)
1. Go to [chat.openai.com](https://chat.openai.com)
2. Use the DALL-E prompts from `PROMPTS.md`
3. Generate each image
4. Download and resize to 1200x630px
5. Optimize with TinyPNG.com

**Cost:** $20/month ChatGPT Plus  
**Time:** 45 minutes for all 8 images  
**Quality:** Very good, unique

---

## Option 2: Canva Templates (Easiest - 1 hour)

### Step-by-Step:
1. Go to [canva.com](https://canva.com) (free account)
2. Create "Custom Size" → 1200 x 630 px
3. Search templates: "tech blog header"
4. Customize with ReviewAI colors:
   - Orange: #F97316
   - Blue: #3B82F6
   - Purple: #8B5CF6
5. Add relevant icons from Canva library
6. Add text overlays (see IMAGE_REQUIREMENTS.md)
7. Download as JPG (optimized)

**Cost:** Free (or $13/month for Pro)  
**Time:** 1 hour for all 8 images  
**Quality:** Good, professional

### Recommended Canva Searches:
- "data analysis blog header"
- "shopping guide banner"
- "tech review header"
- "warning alert banner"

---

## Option 3: Stock Photos + Editing (Budget - 2 hours)

### Free Stock Photo Sites:
1. [Unsplash.com](https://unsplash.com)
2. [Pexels.com](https://pexels.com)
3. [Pixabay.com](https://pixabay.com)

### Search Terms:
- "online shopping data"
- "fake review detection"
- "product research"
- "return package"
- "AI analysis"
- "shopping mistakes"

### Editing:
1. Download high-res images
2. Use Canva or Photopea (free Photoshop alternative)
3. Resize to 1200x630px
4. Add text overlays
5. Add ReviewAI branding
6. Export as JPG

**Cost:** Free  
**Time:** 2 hours for all 8 images  
**Quality:** Good, but less unique

---

## Temporary Solution: Use Placeholder

Until you create custom images, use the existing OG image:

```mdx
image: "/og-image.png"
```

This will work but won't be optimized for each specific blog post.

---

## Image Optimization Checklist

After creating images:

1. ✅ Resize to exactly 1200x630px
2. ✅ Compress with [TinyPNG.com](https://tinypng.com) (target <200KB)
3. ✅ Rename to match blog post (e.g., `fake-reviews-detection.jpg`)
4. ✅ Upload to `/public/blog/` directory
5. ✅ Test social sharing preview:
   - [Twitter Card Validator](https://cards-dev.twitter.com/validator)
   - [Facebook Debugger](https://developers.facebook.com/tools/debug/)
   - [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)
