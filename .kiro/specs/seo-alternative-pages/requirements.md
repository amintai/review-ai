# SEO Alternative Pages - Requirements Document

## 1. Executive Summary

**Feature Name:** SEO Alternative Pages (Fakespot & ReviewMeta)  
**Priority:** P0 (Ship Now) - Per Roadmap Item #3  
**Timeline:** 7-month window closing (Fakespot shut down July 2025)  
**Strategic Value:** Capture orphaned users from defunct competitors


### Problem Statement
Fakespot shut down in July 2025, leaving millions of users searching for alternatives. ReviewMeta has reliability issues. There's a 7-month window (now February 2026) to capture high-intent organic traffic before competitors dominate this space.

### Opportunity Size
- **Search Volume:** "fakespot alternative" + related queries = estimated 50K+ monthly searches
- **User Intent:** Extremely high - users actively seeking replacement tools
- **Competition Gap:** No dominant player has captured this traffic yet
- **Time Sensitivity:** Window closing as competitors build similar pages

## 2. Competitive Intelligence Summary

### Main Competitors Analyzed:

**RateBud** - Primary competitor with aggressive SEO strategy
- Has alternative pages already live
- 100% free, fast analysis (~10s)
- Chrome + Firefox extensions
- **Gap:** Only fake detection, no purchase decision intelligence

**ReviewMeta** - Legacy player with reliability issues
- Slow (30-60s), outdated UI
- Uses cached results
- **Gap:** No real-time analysis, limited features

**Fakespot** - Defunct (shut down July 2025)
- Millions of orphaned users
- Users loved: A-F grades, browser extension
- **Opportunity:** Capture their user base

2. Dual scoring (Trust + Confidence)
3. Fit Analysis (Perfect For / Avoid If / Deal Breakers)
4. Buyer Psychology insights
5. Theme Clustering
6. Persona-Based Verdicts (P1 roadmap)
7. Risk Scoring Layer (P1 roadmap)
8. Product Comparison Mode (P2 roadmap)
9. Similar Product Recommendations (P2 roadmap)

**Positioning:** AI Shopping Decision Copilot (not just fake review detector)
